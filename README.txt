run via ex3.py file (if tested in the 'planet.cs.biu.ac.il' server skip to section 4b):

1. open cmd (Command Shell) or Linux Shell or  in the location dir of the file: ex3.py

2. make sure python is insatlled (latest 3. version preffered):
	a. in cmd, type: python --version
	b. in Linux Shell, type: python3 --version

3. make sure packages are insatlled by typing:
	pip install hexalattice matplotlib

4. run the ex3.py file and add the input 'Elec_24.csv' file for example:
	a. for cmd, type:
			python ex3.py Elec_24.csv
	b. for Linux Shell, type:
			python3 ex3.py Elec_24.csv

two new files will be added to the folder:
1. SOM.png
2. Municipalities_distribution_in_SOM.txt

*note: the figure might not be displayed in Linux but the SOM.png file will display it
